# Neueda Room1 Training Materials

## Overview

This repository contains training materials prepared by **Neueda** for use in **Room1**, one of the dedicated training rooms used for onboarding and upskilling program participants (trainees).

The content in this repository is intended to support structured, hands-on learning sessions delivered as part of Neueda's training programs.
